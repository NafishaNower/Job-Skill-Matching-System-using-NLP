# Job-Skill-Matching-System-using-NLP
# End-to-End NLP Pipeline for Job–Skill Matching

This project implements an end-to-end Natural Language Processing (NLP) pipeline to extract, normalize, and match skills between job descriptions and candidate resumes.

The system combines text preprocessing, skill ontology-based normalization, and multi-layer matching (text extraction, co-occurrence-based recommendation, and hybrid scoring) to generate interpretable match scores along with identified skill gaps.

Key Features:

* Skill extraction from unstructured text
* Semantic normalization using embeddings
* Job–skill matching with precision, recall, and F1 evaluation
* Explainable outputs (matched, missing, and additional skills)

This project demonstrates applied NLP, data preprocessing, and ML evaluation techniques with a focus on real-world recruitment use cases.
